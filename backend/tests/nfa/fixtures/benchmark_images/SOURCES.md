benchmark (`tests/nfa/test_vision_benchmark.py`) and the comprehensive
model comparison (`docs/benchmark/run_benchmark.py`). All images are
from Wikimedia Commons under open licenses that permit redistribution.

## Usage Notes

All images are thumbnails (max ~1280 px long edge) downloaded via the
Wikimedia Commons API. They were selected for photographic category
coverage: golden-hour landscape, macro close-up, night cityscape,
black-and-white portrait, and autumn forest scene.

**For publication:** include the attribution text from the table above
next to each image. CC BY-SA requires share-alike; CC BY requires
attribution only; CC0 requires nothing but attribution is good practice.
